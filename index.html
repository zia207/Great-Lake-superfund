About this data. NPL site list is a curated, substantially expanded set of real EPA Superfund National Priorities List sites in the Great Lakes basin (51 sites), compiled from EPA Superfund program records and Federal Register NPL rulemakings — not the complete SEMS extract (Michigan alone carries 66 active NPL sites statewide). The Areas of Concern layer (30 locations) reflects EPA's Great Lakes AOC program roster. Site and AOC coordinates are approximate centroids.

Watershed boundary: traced directly from the Great Lakes Watershed Boundary KMZ you uploaded (NY State GIS Gateway), simplified from ~21,600 vertices to ~326 for browser rendering. A handful of Gary/East Chicago, Indiana sites (Ninth Avenue Dump, MIDCO II, Lake Sandy Jo, Grand Calumet River AOC) fall just outside this polygon's edge near the heavily engineered Calumet shoreline — flagged individually rather than silently included or dropped.

State boundaries: the second uploaded file (cb_2013_02_anrc_500k.kmz) turned out to contain Alaska Native Regional Corporation boundaries, not Great Lakes state lines — an unrelated Census Bureau layer with a similar filename pattern. In its place this map uses authoritative U.S. Census Bureau state cartographic boundaries (via the us-atlas TopoJSON distribution) for Minnesota, Wisconsin, Illinois, Indiana, Michigan, Ohio, Pennsylvania, and New York. If you have the correct state-boundary file, upload it and it can be swapped in directly.

Verify any site: search it by name in EPA's Superfund site search or see the current list of Great Lakes AOCs.
